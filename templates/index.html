Adri-rollcall is a Web App to store records of Entering/Exiting. To save the record, first, anyone has to have Profile. The Navbar button 'New Person' bring the Admin to the section of creating profile for people. After that, the Navbar button 'Enter/Exit',assist the Amin to save track of them. FrontEnd:
  1. HTML + CSS
  2. BootStrap 5
  3. javascript + jquery
BackEnd:
  1. Python
  2. Flask
  3. Firebase Google Auth
  4. MongoDB Atlas
